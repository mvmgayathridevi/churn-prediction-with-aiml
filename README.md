# 📊 Customer Churn Prediction & Revenue Risk Analytics System

## 📌 Project Overview
This project focuses on predicting customer churn using machine learning and analyzing key business factors that influence customer retention. It helps identify high-risk customers and estimate potential revenue loss.

---

## 🎯 Objective
- Predict whether a customer will churn or stay
- Identify key factors affecting churn
- Assign churn probability scores to customers
- Support business decisions for customer retention

---

## 📂 Dataset
Telecom customer dataset with 7,000+ records including:
- Contract type
- Monthly charges
- Tenure
- Payment method
- Internet services

---

## ⚙️ Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Machine Learning (Random Forest)

---

## 📊 Process Followed
- Data cleaning and preprocessing  
- Handling missing values  
- Feature encoding  
- Exploratory Data Analysis (EDA)  
- Model training and evaluation  
- Churn probability prediction  

---

## 📈 Key Insights
- Month-to-month contracts have highest churn  
- Higher monthly charges increase churn risk  
- Competitor offers strongly influence churn  
- Short tenure customers are more likely to leave  

---

## 💰 Business Impact
- Identified ~₹1,03,000+ revenue at risk  
- Built high-risk customer segmentation  
- Helps improve customer retention strategies  

---

## 🤖 Model Performance
- Accuracy: ~79%  
- Churn probability scoring system implemented  
- High-risk customers identified using threshold > 0.7
